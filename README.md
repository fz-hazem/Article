## 🚀 Getting Started

### Prerequisites
To run this program, you will need a C++ compiler installed on your system. 
* **Windows**: [MinGW](https://www.mingw-w64.org/) (GCC) or MSVC via Visual Studio.
* **Linux**: `g++` (usually available via `sudo apt install build-essential`).
* **Mac**: Clang (available via Xcode Command Line Tools: `xcode-select --install`).

### Compilation
Open your terminal or command prompt, navigate to the directory where you cloned or downloaded the repository, and run the following command to compile the code:

```bash
g++ article.cpp -o article
